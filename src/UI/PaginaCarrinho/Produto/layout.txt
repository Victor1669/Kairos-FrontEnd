Este layout organiza os detalhes de cada produto de forma modular, com uma estrutura que se compacta e reorganiza conforme o espaço disponível na tela.

Organização Visual
Desktop (Telas largas)
O layout mantém a estrutura básica com elementos alinhados horizontalmente.
| Imagem + Nome | Quantidade | Preço | Remover |

Telas Médias (Abaixo de 1175px)
O layout assume uma estrutura de grade (grid) que empilha as informações em blocos.
— Imagem + Nome —
— Qtd | Preço | Total —
— Botão Remover —

Mobile (Telas estreitas)
O layout mantém a estrutura de grade anterior, mas altera a orientação da imagem e reduz os controles.
— Imagem (topo) | Nome —
— Qtd | Preço | Total —
— Botão Remover —

Resumo do Comportamento
Em telas grandes: A visualização é linear. O nome do produto e a imagem ficam lado a lado, com os controles de quantidade e preço organizados em sequência.

Em telas médias: O layout se transforma em uma grade de 3 colunas, agrupando as informações logicamente (identidade, valores e ações) para facilitar a leitura.

Em dispositivos móveis:

Imagem: O alinhamento muda de horizontal para vertical (a imagem vai para cima do nome).

Controles: Os botões de quantidade diminuem de tamanho para otimizar o espaço.

Fonte: O tamanho do texto é reduzido para garantir que tudo caiba confortavelmente em telas pequenas.

Estilização: O produto é exibido dentro de uma caixa com bordas arredondadas e espaçamento interno, garantindo destaque individual para cada item na lista.
